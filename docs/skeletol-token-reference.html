Surface Layers

Three depth tiers. --ink is the void. --panel lifts content. --plum marks active/elevated states and chrome headers. Depth through lightness, not shadow.

Text Hierarchy

--cream for headings and primary body. --muted for secondary, captions, timestamps. --gold for links, labels, and emphasis. Never use pure white.

Borders & Dividers

Subtle: rgba(157,143,130,0.2). Chrome: rgba(157,143,130,0.35). Active: --gold solid. Borders are structural, not decorative.

Shadows

Minimal. Card: 0 2px 8px rgba(10,8,12,0.4). Elevated: 0 4px 16px rgba(10,8,12,0.6). Surface tiers do most of the depth work.

☽ Depth

Surface Hierarchy

Depth through lightness, not shadow. Three tiers. Higher = lighter = closer to viewer.

Layer 0: Base --ink · #1A1520
Layer 1: Raised --panel · #241E2A
Layer 2: Elevated --plum · #3D2845

Radius Rules

2px for buttons, badges, small elements. 4px for cards, panels, chrome windows. 8px for hero containers or full-bleed sections. Never round, never sharp-zero.

Border Weight

Always 1px. No thick decorative borders. Chrome windows and section frames use --border-chrome. Subtle containers use --border-subtle.

Usage Rules

One glyph per section max. Use as eyebrow prefix or centered divider ornament. ◈ is the primary brand mark. ⚓ for nav/anchor contexts. Others rotate freely.

Rendering

Set in --muted or --gold only. Never cream (too loud). Size matches surrounding text or one step up. No animation on glyphs.

☼ Guardrails

System Rules

Non-negotiable constraints. If something violates these, it doesn't ship.

First person only. "I build systems" not "Tony builds systems." Byline credit is fine; body copy is always I/my/me.

No progress signage. No "coming soon," "under construction," "TBD," "being built." Sparse pages are fine. Apology copy is not.

Clarity-first for new visitors. A stranger should understand what I do within 5 seconds of landing. Atmosphere supports clarity, never obscures it.

No generic "creative" framing. Lead with systems, coordination, music, execution under ambiguity. The creative work supports the story of judgment, not the other way around.

Chrome is structure, not costume. W98 motifs frame content. If removing the chrome makes the content harder to read, the chrome is doing its job. If removing it makes no difference, cut it.

Extend frame.w0rk, don't reinvent. This system inherits from Theme System v1 and the existing OS HTML asset. New decisions should feel like the same hand, not a reboot.
